## Project Pulse

Mona's Project Pulse is a lightweight static dashboard exercise that teaches agent-led workflows. Learners use the Orchestrator, Planner, Designer, and Coder agents (defined under .github/agents/) via the GitHub Copilot CLI to plan, design, and implement a small dashboard showing project name, owner, status, recent activity, and priority. The exercise emphasizes delegation, file-scoped work, and producing a runnable app in `app/` with a `.vscode/launch.json` to open `app/index.html`.
